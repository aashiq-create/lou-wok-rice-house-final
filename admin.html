Checkout & Payments

Controls the cart/checkout on the live site. Until your Square account and a payment endpoint are connected, checkout collects the customer's details and places the order as “pay at pickup.” Add the Square IDs below when ready to enable card payment.

Square Payment (fill when ready)

Card data is collected by Square's secure hosted fields — your site never sees raw card numbers. The backend URL receives a payment token (not the card) and charges it via Square's API. Leave blank to keep “pay at pickup” mode.

Deploy to Production

Publish saves cms-data.json to GitHub — index.html is never overwritten. The live site reads this file fresh on every page load, so your changes appear at www.louwok.com on the next refresh — no rebuild wait.
